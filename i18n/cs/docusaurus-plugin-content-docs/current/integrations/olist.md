# O List

<img src="/img/logos/o_list.png" alt="O List logo" width="150" />

`OFeed` je v aplikaci plně podporovaný a je možné ho využít jako zdroj dat. Stačí načíst data na začátku a vzniklé změny na prezentaci jsou automaticky načteny a zobrazeny. Startovka nebo výsledky jsou tak stále aktuální. Po připojení vyčítací krabičky je možné vložením čipu zobrazit závodníka, kterému čip patří (pokud byl nalezen).

Další informace o aplikace najdeš [zde](https://stigning.se/o_list/help_en.html).
