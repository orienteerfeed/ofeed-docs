# Automatický komentátor

V záložce `Výsledky` je možné pomocí ikony reproduktoru zapnout **automatického** komentátora, který hlásí nově **vedoucí** závodníky. Jde zatím o **experimentální** funkci, která ještě vyžaduje **doladění** (například rozlišení mezi doběhl a doběhla).
