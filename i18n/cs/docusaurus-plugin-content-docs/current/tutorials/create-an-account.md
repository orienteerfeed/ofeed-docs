# Vytvoření účtu
Vytvoření účtu a užívání je **bezplatné**. Přes tlačítko `Přihlásit` v pravém horním rohu a výběrem `Ještě nemáš svůj účet? Zaregistruj se` se dostaneš na založení účtu.

![Visualize Data](/img/tutorials/create-an-account-signin.png)

Vyplň formulář

![Visualize Data](/img/tutorials/create-an-account-form.png)

a po potvrzení ti do emailu dorazí potvrzování email o úspěšně registraci. Pak se stačí přihlásit a můžeš rovnou založit svůj první [nový závod](/docs/tutorials/create-a-new-event).
