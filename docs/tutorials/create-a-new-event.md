# Create a New Event

The first step is to create a new event that will store all your data. You need to be logged in, and then you can create a new event by clicking the `+ Create event` button.

![Create button](/img/tutorials/client/create_event.png)

and filling out the form.

![Event form](/img/tutorials/client/create_event_form.png)

## Generate event password

The password is used to update event details. The **event ID** is also **required** to set up the connection in, for example, `OChecklist` or the `QuickEvent service`.

![Event form](/img/tutorials/client/settings_password.png)

:::info[Warning]
The password is valid for 7 days. Keep an eye on the countdown and regenerate the password if needed.
:::
