# Create an Account

Create a new user and access all available features for `free`. Simply use the `Sign Up` option in the upper right corner, then select `Don't have an account? Sign Up`.

![Sign in](/img/tutorials/client/create-an-account-signin.png)

Fill in the form and confirm.

![Create an account](/img/tutorials/client/create-an-account-form.png)

Next, check your email inbox and confirm the email address you provided.
![Confirmation email](/img/tutorials/client/create-an-account-confirmation-email.png)

After that, you can log in and [Create a new event](/tutorials/create-a-new-event) as desired.
