# O List

<img src="/img/logos/o_list.png" alt="O List logo" width="150" />

`OFeed` is fully supported in the application and can be used as a data source. Simply load the data at the beginning, and any changes in the presentation are automatically loaded and displayed. The start list or results are thus always up to date. After connecting the reading device, inserting a card makes it possible to display the competitor to whom the card belongs (if found).

More details are available [here](https://stigning.se/o_list/help_en.html).
