# Joja

## Apps

Popular mobile apps for organising events on mobile devices.

### SI-Droid Event

<img src="/img/logos/sidroid.png" alt="Si Droid logo" width="150" />

Mobile app for organising small events on your mobile device.

[Download](https://play.google.com/store/apps/details?id=se.joja.sidroid.event.full&hl=cs&pli=1)

### O-Pal

<img src="/img/logos/opal.png" alt="O-Pal logo" width="150" />

Improved version of `SI-Droid Event` with startlist and more new features.

[Download](https://play.google.com/store/apps/details?id=se.joja.opal.event&hl=cs)

## SI-Droid Connector

<img src="/img/logos/connector.png" alt="Connector logo" width="150" />

A mobile app published as an open-source [project](https://github.com/orienteerfeed/ofeed-sidroid-connector) that handles XML uploads to the OFeed platform using the `Results Service` running in `SI-Droid Event` or `O-Pal`. This allows the readout process of your event to be managed easily on your mobile device.
