# Contribute

`OFeed` is open source software. As such, we welcome contributions and help from any and all interested people.

## Providing Feedback

We don't know of a better way to test than by using it during events. It doesn't matter if it's training or a race. Every use on real data is the best way to verify functionality. It also provides a lot of feedback, which we are very interested in. We would be very happy if you would write your findings on Discord or create an issue on Github.

## Writing/Editing Documentation

If you find any inaccuracies or something in the documentation is unclear, we would appreciate your feedback. You can use the button at the bottom of the page to open the relevant file.

## Translating
